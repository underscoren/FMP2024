# FMP 2024 website

Made with: 
- Tooling: vite-create
- Framework: Svelte, SvelteKit
- Language: TypeScript, Sass
- CSS framework: Bulma

## Developing

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

```bash
npm run build
```
